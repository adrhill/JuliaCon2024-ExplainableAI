Methods: input sensitivity

using ExplainableAI
using VisionHeatmaps

analyzer = Gradient(model)
expl = analyze(input, analyzer)
heatmap(expl)

Methods: GradCAM

using ExplainableAI
using VisionHeatmaps

analyzer = GradCAM(model[1], model[2])
expl = analyze(input, analyzer)
heatmap_overlay(expl, img)

Methods: LRP

using RelevancePropagation
using VisionHeatmaps

analyzer = LRP(model, EpsilonPlusFlat())
explanation = analyze(input, analyzer)
heatmap(explanation)

What‚Äôs new: XAIBase.jl

Interface definition:

  1. method has to be a subtype of AbstractXAIMethod
  2. method has to implement analyze function
  3. method has to return Explanation

Benefits

  • automatically compute heatmaps using VisionHeatmaps.jl and TextHeatmaps.jl
  • composable with e.g. input augmentations from ExplainableAI.jl
  • dependency free
  • perfect for package extensions!
